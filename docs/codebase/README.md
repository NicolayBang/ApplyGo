# Codebase Reference

This directory contains C4 code-level documentation for the implemented ApplyPilot
Milestone 1 codebase.

The pages describe modules, classes, functions, contracts, tests, workers, migrations,
and runtime relationships. They are implementation references, not architecture
authority. If a page conflicts with the implemented code or the locked architecture,
follow the authority order in `AGENTS.md` and correct the documentation.

## Maintenance

- Update affected pages in the same pull request as architecture-related code changes.
- Describe only implemented behavior.
- Mark future behavior as **Planned** or **Not Implemented**.
- Treat counts and line references as snapshots that may change as the code evolves.
